## ✨ Vision du Projet
**Babyfoot.io** est une solution moderne conçue pour organiser, suivre et documenter des tournois de babyfoot avec une expérience utilisateur fluide et une architecture backend robuste.

> **Note de l'auteur :** Ce projet a été réalisé dans le cadre d'un test technique.

## 🚀 Fonctionnalités Clés
- **Gestion des Tournois** : 
- **Architecture Modulaire** : Backend NestJS.
- **Documentation Automatisée** : Intégration complète de Swagger et Compodoc.
- **Feedback UX** : Système de notifications (Toasts) et design responsive.

---

## 🛠 Stack Technique

### Frontend
- **Framework** : Nuxt 3 (Vue.js 3)
- **Styling** : Tailwind CSS
- **Notifications** : Vue-Sonner

### Backend
- **Framework** : NestJS (Node.js)
- **ORM** : Sequelize (MySQL)
- **Validation** : Class-Validator & Class-Transformer
- **API Doc** : Swagger UI
